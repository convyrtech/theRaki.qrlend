## 2. Палитра и визуальная идентичность

### Концепция: "Dark Gzhel"

Не просто тёмный сайт и не просто народный — синтез двух миров:
- **Тёмный премиум** как основа (как в лендинге)
- **Народный орнамент** как декоративный элемент (отсылка к гжельскому мотиву и маскоту-ракше)

### Цветовая система

| Роль | Цвет | HEX |
|------|------|-----|
| Основной фон | Тёмный сине-зелёный | `#0b3237` |
| Тил-акцент | Средний тил | `#51a8af` |
| Премиум | Золото | `#be9667` |
| Народный акцент | Красный (декоративно!) | `#e10600` |
| Текст | Белый / кремовый | `#ffffff` / `#f5f0eb` |
| Карточки | Белый с opacity | `rgba(255,255,255,0.08)` |

### Принципы применения

- `#e10600` — ТОЛЬКО в орнаментах, народных узорах, декоративных разделителях. **Никогда** на кнопках и интерактивных элементах.
- Золото `#be9667` — цена, спецпредложения, акценты премиального уровня
- Тил `#51a8af` — основной CTA-цвет (кнопки, ссылки)
- Маскот (краб из машинной обклейки) — уместен в тёмном контексте как живой элемент бренда

### Визуальные отсылки

- Japandi dark aesthetic (2025-2026 тренд)
- Materia prima luxury restaurants (Barcelona, NYC)
- Тонкие орнаментальные линии à la Fabergé
- Профессиональные фото живых раков как главный герой

## 3. Типографика

### Display: Cormorant Garamond

```css
@import url('https://fonts.googleapis.com/css2?family=Cormorant+Garamond:ital,wght@0,300;0,400;0,600;1,300;1,400&display=swap');
```

- Вес 300 на тёмном фоне = мгновенный сигнал люкса
- Полная кириллица
- Бесплатно, Google Fonts
- Italic для tagline и цитат

**Применение:**
- H1 (hero): `font-size: clamp(48px, 8vw, 96px)`, `font-weight: 300`, tracking `-2px`
- H2 (секции): `font-size: clamp(32px, 5vw, 56px)`, `font-weight: 300`
- Цитаты / tagline: `font-style: italic`, `font-weight: 300`

### Body: PT Sans

```css
@import url('https://fonts.googleapis.com/css2?family=PT+Sans:wght@400;700&display=swap');
```

- Создан специально для кириллицы
- Параллобластность: отлично читается на тёмном фоне
- Знакома русскоязычной аудитории — нет когнитивного барьера

**Применение:**
- Параграфы: `font-size: 16-18px`, `line-height: 1.7`, `font-weight: 400`
- Подписи / метаданные: `font-size: 13-14px`, `color: rgba(255,255,255,0.5)`

### Альтернатива body: Neue Haas Grotesk / TT Norms Pro

Если бюджет позволяет — платные, но дают более европейский премиальный вид.

## 4. UX и структура сайта

### Концепция: Hybrid Brand Story + Smart Routing

Пользователь сначала погружается в историю, затем совершает действие.

### Структура главной страницы (9 секций)

```
1. HERO
   ├── Полноэкранное видео или фото (раки в кастрюле / краб)
   ├── Headline: "The Raki. Живые. Только так."
   ├── Sub: "Раки и морепродукты с доставкой по Москве"
   └── Split CTA: [Заказать домой] / [Корпоративный заказ]

2. ROUTING SPLIT (появляется после скролла)
   ├── B2C: карточки меню → корзина
   └── B2B: кейсы, объёмы, контакт менеджера

3. SOCIAL PROOF / TRUST
   ├── "Никогда не замораживают" — объяснение процесса
   ├── НТВ-упоминание (медиа присутствие)
   └── Счётчики: 7 лет / 15 лет опыта шефа / N заказов

4. ПРОДУКТЫ (сезонное меню)
   ├── Живые раки (категории по размеру/приготовлению)
   ├── Морепродукты (краб, мидии, креветки)
   └── Сеты / Корпоративные заказы

5. ИСТОРИЯ БРЕНДА
   ├── Шеф с 15-летним опытом
   ├── "Варим живых прямо перед доставкой"
   └── Фото кухни / шефа / процесса

6. КАК ЭТО РАБОТАЕТ
   ├── 1. Заказ → 2. Приготовление живых → 3. Доставка горячими
   └── Время доставки, зоны

7. ОТЗЫВЫ
   └── Авторизованные отзывы (не анонимные) + фото

8. ПРИЛОЖЕНИЕ
   ├── App Store / RuStore / APK
   └── QR-код (из нашего deploy/)

9. FOOTER
   ├── Контакты: +7 (980) 888 05 88
   ├── theraki.ru
   └── Навигация
```

### Мобильный UX

- **Sticky CTA бар** внизу: `[Заказать]` (всегда видна кнопка)
- Hero: нет автоплей-видео (battery/data сбережение)
- Кнопки: min-height 56px (fat finger friendly)
- Изображения: WebP с JPEG fallback

### Desktop UX

- Hero: fullscreen видео (автоплей, muted, loop)
- Hover-эффекты на карточках продуктов (subtle glow)
- QR-код в секции "Приложение"
- Плавный скролл, parallax для hero-фото

## 5. Психология и позиционирование

### Концепция: "Quiet Superiority"

Не кричать о премиуме — демонстрировать его через детали.

### Главный психологический триггер

> **"Никогда не замораживают"**

Это не просто факт — это радикальная честность. В мире где 99% конкурентов используют заморозку, это утверждение:
- Создаёт немедленное доверие
- Дифференцирует от всех конкурентов
- Апеллирует к ценностям состоятельного клиента (качество > экономия)

### Что убрать из Hero

- ❌ Кешбэк (это сигнал скидочного магазина, не премиума)
- ❌ "Раки для гурманов" — банально
- ❌ Прайсы в hero-секции

### Что добавить

- ✅ "Никогда не замораживают" — как badge или подзаголовок
- ✅ Дата основания / история (доверие через возраст)
- ✅ B2B-путь из hero (корпоративный сегмент — высокий чек)
- ✅ Шеф с именем (персонализация через человека)

### B2B vs B2C сегментация

| Параметр | B2C | B2B |
|----------|-----|-----|
| Триггер | "Угости семью лучшим" | "Впечатли партнёров" |
| CTA | "Заказать сейчас" | "Обсудить меню" |
| Контент | Меню, доставка, приложение | Кейсы, объёмы, менеджер |
| Ценовая коммуникация | Открытые цены | "Специальные условия" |

## 6. Тренды 2026

### Релевантные тренды для премиум ресторанного сегмента

| Тренд | Применение |
|-------|-----------|
| **Cinematic Dark Minimalism** | Тёмный фон, cinematic-фото, большие шрифты, воздух |
| **Editorial Typography** | Cormorant Garamond, крупные display-заголовки, serif для luxury |
| **Micro-interactions** | Hover на кнопках, cursor-tracking glow, smooth scroll |
| **No-popup policy** | Никаких всплывающих окон. Анонс — только inline |
| **Mobile-first** | 70%+ трафика с мобильных, sticky CTA |
| **Video hero** | Короткий loop-видео (5-10 сек) вместо статичного фото |
| **Social proof integration** | Отзывы с фото и именами, не анонимные звёзды |
| **Speed = Luxury** | LCP < 1.5 сек, никаких тяжёлых библиотек |

### Что НЕ делать (антитренды 2026)

- ❌ Chatbot popup в правом углу
- ❌ Cookie banner во весь экран
- ❌ Счётчики обратного отсчёта "Осталось 3 порции!"
- ❌ Мигающие акционные плашки
- ❌ Google Analytics (GDPR + не работает хорошо в РФ)

### Аналитика

- **Yandex.Metrika** — вебвизор, тепловые карты, бесплатно, работает в РФ
- Цели: клик на телефон, клик на приложение, клик на меню

### Производительность

- Шрифты: только Google Fonts (CDN, кэш), не самохост на первом итоге
- Изображения: WebP, lazy load, размеры указывать явно
- CSS: инлайн критический, остальное defer
- JS: минимум, никаких фреймворков (vanilla, как лендинг)

## 10. Референс-анализ: что украсть у лучших

### Проанализированные сайты

| Сайт | Тип | Рейтинг | Ключевая находка |
|------|-----|---------|-----------------|
| Quay Restaurant (Австралия) | Awwwards SOTD | ⭐⭐⭐⭐⭐ | SVG stroke-анимация логотипа при загрузке |
| Eleven Madison Park (NYC) | 3 звезды Мишлен | ⭐⭐⭐⭐⭐ | Gallery hero + один CTA + storytelling |
| Seabird London | Premium rooftop | ⭐⭐⭐⭐ | Video hero + persistent CTA в хедере |
| Beefbar (глобальная сеть) | Luxury brand | ⭐⭐⭐⭐⭐ | Scroll color transitions между секциями |
| Клёво (Москва, Новиков) | Прямой конкурент | ⭐⭐ | Tilda-шаблон, слабее чем может быть |
| Le Bernardin (NYC) | 3 звезды Мишлен | ⭐⭐⭐⭐ | Меню по методу приготовления |

### Что украсть (конкретные техники)

#### 1. SVG stroke-анимация логотипа (Quay)
```css
/* Логотип "рисуется" при загрузке — 2-3 секунды */
.logo-path {
    stroke-dasharray: 1000;
    stroke-dashoffset: 1000;
    animation: draw 2.5s ease-in-out forwards;
}
@keyframes draw {
    to { stroke-dashoffset: 0; }
}
```
Адаптация: буква "R" или силуэт рака рисуется при загрузке.

#### 2. Rotating gallery hero (Eleven Madison Park)
- 6-8 профессиональных фото (раки, шеф, процесс, морепродукты)
- Автокарусель 5 сек, плавный crossfade
- НЕТ текста поверх фото в первые 2 сек — дать изображению дышать
- Потом fade-in заголовка

#### 3. Background color transitions при скролле (Beefbar)
```css
/* Секции плавно меняют цвет фона при скролле */
transition: background-color 1000ms cubic-bezier(0.25, 1, 0.5, 1);
```
- Hero: `#0b3237` (тёмный)
- Trust-секция: `#1a4550` (чуть светлее)
- Меню: `#0b3237` (снова тёмный)
- История: `rgba(255,255,255,0.05)` overlay (почти чёрный)

#### 4. Centered logo + split navigation (Beefbar)
```
[Меню]  [О нас]  [LOGO]  [Доставка]  [Заказать →]
```
Элегантнее стандартного левый-логотип + правое-меню.

#### 5. Uppercase 2px letter-spacing для nav и CTA (Beefbar)
```css
.nav-link, .btn { letter-spacing: 2px; text-transform: uppercase; }
```
Мгновенно добавляет "ювелирности".

#### 6. Storytelling по методу приготовления (Le Bernardin)
Вместо "Меню":
- **Живые** — раки, привезённые живыми
- **Варёные** — сваренные прямо перед доставкой
- **Жареные** — золотистые, хрустящие

#### 7. Context-aware модал заказа (Quay)
- Мобильный: drawer снизу (нативный feel)
- Десктоп: панель справа (не уходит со страницы)

### Слабость конкурентов (эксплуатируем)

**Клёво** (главный московский конкурент в premium seafood):
- Построен на Tilda — шаблон, заметно
- Тил `#4195AF` + белый — светлый, не тёмный
- Нет визуальной дифференциации от сотен других ресторанов Новикова

**The Raki выигрывает через:**
- Custom код (не Tilda) — уже другой уровень
- Тёмный дизайн vs их светлый — визуально контрастируем
- "Никогда не замораживают" vs их стандартная подача
- История с 2017, шеф с лицом — Клёво безличен
